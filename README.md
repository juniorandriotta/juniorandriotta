## ⚡ Sobre o meu trabalho

Construo arquiteturas escaláveis e soluções ponta a ponta, com foco absoluto em performance, previsibilidade de custos e uma experiência de usuário impecável. 

- 🏗️ **Arquitetura:** SaaS Multi-Tenant, Microserviços e Monólitos Modulares.
